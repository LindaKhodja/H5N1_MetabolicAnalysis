# H5N1_WebServer
Une plateforme web développée pour centraliser et intégrer les différents outils, permettant ainsi à la communauté scientifique d'accéder aux services bioinformatiques, de charger des données, de réaliser des analyses et des modélisations des voies métaboliques du virus, et d’explorer les résultats de manière interactive.


Ajout des fichiers (version finale) dans la branche Céline (attention aux pull)
- ui.R = interface ui
- server.R = fonction liées à l'affichage, clic etc dans l'interface
- global.R = dépendance, imports, variables
- utils.R = fonction (dont parseur)
- app_AnaMetabo.R = pour lancer l'app (il faut juste lancer ce code et être sûr d'avoir tous les autres scripts dans le même dossier)
